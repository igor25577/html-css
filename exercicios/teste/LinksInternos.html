Mecânicas de links internos

Os links internos são links que te levam a um lugar de seu proprio dominio ou melhor dizendo ele apenas muda as paginas de seu site ao contrario do link externo ele n precisa de uma ancoragem ele so precisa do rel="internal" para sinalizar sua origem. Os links internos podem ter pasta dentro da pasta para te direcionar a outra pagina porem essas pra voltar elas são um pouco complicadas sendo necessario o comando "a href="../nome da pasta/nome do arquivo" para fazer o retorno de maneira sem falhas e seguro utiliza o comando target="_self" para sinalizar que é para retornar para a propria pagina
