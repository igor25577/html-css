# Projeto Android
#### Meu Projeto Android, é um projeto de um curso da plataforma CursoEmVideo, onde utilizamos diversos recursos visuais, links com ancoragem para outras paginas, imagens e 100% responsivo, além de Menu iterativo, e um iframe de um video no YouTube nesse site contamos um pouco da historia do surgimento do android com diversos detalhes e curiosidades Abaixo tera imagens para a compreensão visual
